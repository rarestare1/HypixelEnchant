# HypixelEnchant
This script automates the 2 minigames in the enchantment table for hypixel skyblock.
You need NEU mod for this script to work.
# Keybindings
###### F1=Only color game
###### F2=Reload
###### F3=Both sequence and color
###### F4=Close script
# Features
Autofishing and maybe more will come in the future.
